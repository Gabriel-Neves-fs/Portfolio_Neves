# Gabriel Neves | Portfolio

Portfolio pessoal construido com Next.js, TypeScript e Tailwind CSS.

Repositorio: `https://github.com/Gabriel-Neves-fs/Portfolio_Neves`

## Objetivo

Apresentar a transicao de Analista de Suporte Tecnico Pleno para Desenvolvimento Full Stack, conectando experiencia em sustentacao, investigacao tecnica, APIs, logs e integracoes com projetos reais de software.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui style primitives
- Lucide Icons

## Estrutura

- `src/app`: rotas, layout global e pagina principal
- `src/components`: componentes reutilizaveis
- `src/data`: dados locais de projetos, experiencias e skills
- `src/lib`: utilitarios compartilhados

## Scripts

```bash
npm run dev
npm run build
```
